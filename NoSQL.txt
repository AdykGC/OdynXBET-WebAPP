Чтобы сохранить изменения в GitHub после редактирования, выполните следующие шаги:

Добавьте изменения:
Перейдите в терминал и убедитесь, что находитесь в каталоге вашего локального репозитория (OdynXBET-WebAPP).
Если вы внесли изменения в файл ASSIGMENT_1_P_3.html, добавьте их в индекс Git с помощью команды:
bash
Copy code

--------- 1.     git add .
Сделайте коммит:
После добавления изменений в индекс, выполните коммит, чтобы зафиксировать изменения. Замените "Ваше сообщение коммита" на описание ваших изменений:
bash
Copy code
--------- 2.     git commit -m "Ваше сообщение коммита"
Отправьте изменения на GitHub:
После коммита отправьте изменения на GitHub с использованием команды git push:
bash
Copy code
--------- 3.     git push origin LEARN
Если у вас впервые запрашивают учетные данные, введите свое имя пользователя и пароль от учетной записи GitHub.
Теперь ваши изменения должны быть сохранены в репозитории на GitHub. Вы можете проверить обновления на веб-странице репозитория. В дальнейшем для сохранения изменений, вам будет достаточно использовать команды git add, git commit и git push в вашем терминале.
